# Dev Server

// turbo-all

## Başlat
```bash
npm run dev
```

## Durdur
Terminal'de `Ctrl+C` kullan.

## Port
- Varsayılan: http://localhost:3000
- Alternatif: http://localhost:3001 (3000 meşgulse)

## Build
```bash
npm run build
npm run start
```
